我当时做的时候是用了3个干电极


干电极可以替换成湿电极，湿电极波形更加稳定，但是佩戴相对麻烦

干电极买回来后，需要用arduino采集信号，并传输给电脑处理

设置好相应端口号后，运行main函数，按照屏幕上的指示进行信号采集，然后等待一会，做模型训练

训练完后就可以实时识别手势信号了
